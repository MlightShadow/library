#### 逻辑运算符

通常我喜欢称之为逻辑运算符
无论是进行比较或者做一些与,或运算 这类运算符的结果一般用于反馈真假的判断

| 符号 | 名称 |
| ---- | ---- ||  |
| `>`  | 大于     |
| `<`  | 小于     |
| `==` | 等于     |
| `<=` | 小于等于 |
| `>=` | 大于等于 |
| `!`  | 非       |
| `!=` | 不等于   |
| `&&` | 与       |
| `||` | 或       |
| `&`  | 按位与   |
| `|`  | 按位或   |
| `^`  | 异或     |
